# 📸 DQ Blendz — Image Uploads

Drop image files into the folders here and they become part of the website.
Anything inside `public/` is served from the site root, so a file at:

```
public/images/gallery/cut-01.jpg
```

is available on the site at:

```
/images/gallery/cut-01.jpg
```

---

## Where to put what

| Folder            | What goes here                                            |
| ----------------- | --------------------------------------------------------- |
| `images/gallery/` | Photos of cuts for the **Gallery** (fades, lineups, etc.) |
| `images/about/`   | A photo of **DQ / the shop** for the "Barber" section     |
| `images/hero/`    | A background photo for the **top of the page** (optional) |
| `images/logo/`    | A real **logo** file, if/when DQ has one                  |

## How to upload

1. Open the folder above on GitHub (or in your editor).
2. Add / drag your image files in.
3. Commit them to the repo.
4. Tell me they're uploaded and I'll wire them into the site.

## Tips for good results

- **Format:** `.jpg` for photos, `.png`/`.svg` for logos.
- **Size:** ~1200–1600px on the long edge is plenty. Keep files under ~500 KB
  each if you can (compress at e.g. https://squoosh.app) so the site stays fast.
- **Naming:** simple, lowercase, no spaces — e.g. `cut-01.jpg`, `skin-fade.jpg`,
  `dq-portrait.jpg`.
- **Orientation:** gallery looks best with a mix of portrait shots.

Once your files are in, I'll swap them into the Gallery, the Barber section,
and (if you want) the hero background.
